Sur votre page de gestion des attributs, vous afficherez la liste de tous les PropertyAttribute.
Chaque ligne de la liste aura deux boutons : "Modifier" et "Supprimer".
Clic sur "Modifier" :
Ouvre une modale ou une nouvelle page pré-remplie avec les informations de l'attribut.
L'admin change le nom ou le type de données et sauvegarde.
Votre panel envoie une requête PUT à /admin/property_attributes/<id> avec les nouvelles données.
Clic sur "Supprimer" :
Affiche une boîte de dialogue de confirmation : "Voulez-vous vraiment supprimer cet attribut ?".
Si l'admin confirme, votre panel envoie une requête DELETE à /admin/property_attributes/<id>.
Si l'attribut est utilisé, l'API renverra une erreur 409 Conflict avec un message explicatif que vous pourrez afficher à l'admin.
Si l'attribut n'est pas utilisé, l'API le supprimera et renverra un succès.
